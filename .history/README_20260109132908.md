## Technologies Used

- **React Native** - Mobile app framework
- **Expo** - Development platform for React Native
- **Expo Camera** - Camera API for capturing photos
- **Expo Media Library** - Save photos to device

## Prerequisites

Before running this project, make sure you have:

- Node.js installed
- Expo CLI installed (`npm install -g expo-cli`)
- Expo Go app on your mobile device (for testing)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Afaqhaider20/practice-zoom-clone-expo.git
cd practice-zoom-clone-expo
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Scan the QR code with Expo Go app on your device

## Usage

1. Grant camera and media library permissions when prompted
2. Tap "Start Camera" to activate the camera
3. Use the camera controls to take photos
4. Switch between front and back camera as needed

## Scripts

- `npm start` - Start the Expo development server
- `npm run android` - Run on Android device/emulator
- `npm run ios` - Run on iOS device/simulator
- `npm run web` - Run in web browser
